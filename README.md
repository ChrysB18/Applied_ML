# Applied_ML

## Overview
This repository contains a structured implementation of classical machine learning models applied to biomedical data.

## Project Structure
- src/: Core training and evaluation scripts
- notebooks/: Exploratory analysis
- results/: Model outputs and evaluation artifacts

## Methods
Models implemented:
- Logistic Regression
- Support Vector Machine

## Evaluation
Performance assessed using:
- Accuracy
- Precision / Recall
- ROC-AUC
- Confusion Matrix

## Reproducibility
Instructions for running experiments are provided in the training script.
